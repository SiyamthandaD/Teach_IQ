Gemini API: Rate Limits & Usage Costs

A detailed overview of the quotas and pricing for the Gemini 2.5 Flash Preview 05-20 model.

Gemini 2.5 Flash Preview 05-20: Rate Limits

Rate limits are applied per project and are measured across four dimensions: Requests per Minute (RPM), Tokens per Minute (TPM), and Requests per Day (RPD). Exceeding any of these limits will result in a rate limit error (HTTP 429).

Current Rate Limits for gemini-2.5-flash-preview-05-20:

  • RPM (Requests Per Minute): 2,000
  • TPM (Tokens Per Minute): 3,000,000
  • RPD (Requests Per Day): 100,000

Important Note: Preview models often have more restrictive rate limits compared to stable or generally available models. These limits are subject to change as the model progresses.

You can request an upgrade to higher tiers with increased rate limits as your API usage and spending increase. This option usually appears on the API keys page in Google AI Studio for projects that qualify.

Gemini 2.5 Flash Preview 05-20: Usage Costs

The pricing for the Gemini API is generally structured on a per-million-token basis, with separate costs for input and output tokens. For preview models, there's often a free tier before paid usage applies.

Pricing for gemini-2.5-flash-preview-05-20:

Free Tier:

  • Input: Free of charge
  • Output: Free of charge

Paid Tier (per 1M tokens in USD):

Category Cost per 1M Tokens (USD) Notes
Input Price (Text / Image / Video) $0.15
Input Price (Audio) $1.00
Output Price (Non-thinking) $0.60
Output Price (Thinking - if enabled) $3.50 Higher cost when the model's "thinking" capability is active.

Important Considerations:

  • "Thinking" capability: The Gemini 2.5 Flash model now includes a "thinking" capability. If you utilize this, the output pricing for thinking tokens will be higher. When using the standard variant (without the ":thinking" suffix), the model will avoid generating thinking tokens.
  • Context Caching: There are also separate costs for context caching if you use that feature. For Gemini 2.5 Flash Preview, context caching is priced at $0.0375 (text/image/video) and $0.25 (audio) per 1M tokens, and $1.00 per 1,000,000 tokens per hour for storage.
  • Grounding with Google Search: This feature has a free tier for up to 1,500 RPD, then it's $35 per 1,000 requests.
  • Text-to-Speech (TTS) models: If you are using the Text-to-Speech variant of Gemini 2.5 Flash Preview (`gemini-2.5-flash-preview-tts`), the pricing is:
    • Input: $0.50 per 1M text tokens
    • Output: $10.00 per 1M audio tokens
  • Data Usage for Product Improvement: For the Free Tier, your data may be used to improve Google products. For the Paid Tier, this is generally not the case.
